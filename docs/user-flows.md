# Fluxos do Usuário (User Flows)

Este documento descreve, de forma simples, como um usuário interage com a
plataforma desde o primeiro acesso até o pós-encontro.

## 1. Cadastro do Usuário
1. Usuário acessa a plataforma
2. Cria uma conta
3. Preenche informações básicas:
   - Nome
   - País de origem
   - Idiomas
   - Tipo de viagem (intercâmbio, turismo, estudo)

## 2. Cadastro da Viagem
1. Usuário cadastra uma nova viagem
2. Informa:
   - Cidade e país de destino
   - Datas de início e fim
   - Aeroporto
3. Marca o status da viagem como planejada ou confirmada

## 3. Recebimento de Matches
1. A plataforma analisa viagens com datas e locais compatíveis
2. Usuário recebe sugestões de match
3. Cada match mostra:
   - Por que foi sugerido
   - Datas e cidade em comum
   - Tipo de conexão (cultural, amizade, etc.)

## 4. Agendamento de Encontro
1. Usuários conversam dentro da plataforma
2. Escolhem um local público
3. Definem data e horário
4. Confirmam o encontro

## 5. Encontro Real
1. Encontro acontece fora da plataforma
2. Usuários seguem recomendações de segurança

## 6. Pós-Encontro
1. Usuários confirmam se o encontro aconteceu
2. Podem deixar uma avaliação ou feedback
3. A reputação do usuário é atualizada
