# Funcionalidades do Projeto

Este arquivo descreve as funcionalidades planejadas para o aplicativo,
começando pelo MVP (mínimo produto viável) e evoluindo para versões futuras.

## MVP (Versão Inicial)
- Cadastro de usuário
  - Nome, país de origem, idiomas, tipo de viagem
- Cadastro de viagem
  - Destino, datas, aeroporto
- Match inteligente baseado em datas e locais
- Agendamento de encontro seguro
- Checklist básico de segurança

## V2 (Versão Intermediária)
- Score de compatibilidade explicável
- Reputação temporal (expira com o tempo)
- Filtro por tipo de encontro: cultural, amizade, networking, romântico
- Micro-janelas de disponibilidade para encontros rápidos

## V3 (Versão Avançada)
- Plano B automático em caso de cancelamento
- Modo “Viagem Sensível” (primeira viagem/intercâmbio)
- Avaliações pós-encontro
- Sistema de notificações inteligentes

## Notas
- Cada versão será construída de forma incremental
- O foco inicial é garantir segurança e planejamento
- Funcionalidades complexas podem ser simuladas no MVP
